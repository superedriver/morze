# Converting Morse code from space button
This app converts morse signals from space button to letters.

The app is based on:
- [ReactJS](https://reactjs.org) 
  
To use it:
 - clone the repo and go to repo folder
 - execute commands:
 
`$ npm install`

`$ npm start`
